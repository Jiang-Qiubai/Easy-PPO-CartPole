# 如何使用这个代码来学习PPO算法

1.根据pyproject.toml复现环境

2.打开PPO.py运行代码，即可看到训练结果图片，关闭弹窗即可看到训练结果在测试集上的效果（动画）

3.想对参数有一个清晰的认知可以在PPO.py里面class Config:里面修改，我已经为每个参数增加了注释
